Hardware Recommendations

Choosing the right label printer and barcode scanner for your lab.

Label Printers

Dymo LabelWriter 400

Budget-friendly, Linux/CUPS

Great entry-level option with direct CUPS support on Linux. Uses standard Dymo label rolls.

~$100–200

Brother QL-820NWB

Best value, WiFi + Bluetooth

Wireless printing over WiFi or Bluetooth. Excellent print quality with DK label rolls.

~$150–350

Zebra ZD421

Enterprise-grade, ZPL support

Professional thermal printer with native ZPL command support. Best for high-volume labs.

~$300–600

Barcode Scanners

Any Bluetooth Keyboard-Wedge Scanner

flyRoom works with any standard Bluetooth barcode scanner ($50–100). The scanner appears as a keyboard input — no drivers needed.

  • Supports Code128 barcodes and QR codes
  • Plug-and-play — no driver installation
  • Instant stock lookup from any page

Print Agent (flyprint)

flyRoom includes flyprint, a free desktop print agent that bridges the web app to your local label printer. It runs on Linux and Windows, connects to CUPS or native print services, and pairs with the web app automatically — zero configuration required.
